Straky Games Inc. est une petite organisation de
conception de jeux vidéos créer par $traker (moi même).

Elle consiste à développer des jeux de type Ordinateur ou Téléphone avec
plusieurs langages de programmation (C++, Python, Java, etc...) et/ou
logiciels de développement (Clickteam Fusion, Game Maker Studio, Unity 3D, etc...)

Des beta testeurs serons la pour tester les jeux avant leurs déploiements sur les
différentes plateforme de téléchargement ou d'achat (Steam, Google Play, etc...).
Ils donnerons leurs avis sur les différents jeux testés, ceux-ci serons donc
amélioré avec leurs tests et leurs différentes évaluations.

Mes jeux, dans cette organisation sont disponible
dans l'onglet "Mes Projets" de ce site internet.
